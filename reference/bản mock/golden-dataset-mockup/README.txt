GOLDEN DATASET BUILDER — MOCKUP (AI4Health)
============================================

Đây là bản mockup GIAO DIỆN (không phải app chạy thật) để tham khảo khi xây dựng.

CÁCH MỞ
-------
Nhấp đúp vào file "golden-dataset-mockup.html" — mở bằng trình duyệt
(Chrome / Safari / Edge). Không cần cài đặt, không cần internet.

NỘI DUNG MOCKUP
---------------
1. Màn chọn bác sĩ  — danh sách do Admin tạo sẵn, mỗi người 1 chuyên khoa.
2. Màn nhập liệu    — sidebar 24 loại câu hỏi (badge x/5), form nhập gồm:
     • Câu hỏi
     • Trích dẫn: CHỌN đoạn guideline có sẵn (chunk_id) + liệt kê các ý
     • Câu trả lời mong muốn
   Bấm "Ghi chú cho dev" (góc phải) để xem spec kỹ thuật + schema.
3. Màn Admin        — thống kê, bảng bác sĩ + tiến độ, bản đồ 24 loại, thêm bác sĩ.

LƯU Ý
-----
- 24 loại câu hỏi và dữ liệu bác sĩ trong đây là VÍ DỤ để minh họa,
  sẽ thay bằng dữ liệu thật.
- Bấm thử được: chọn bác sĩ -> nhập -> Lưu -> tiến độ tự cập nhật.
